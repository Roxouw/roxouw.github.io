# Project Context

## Objective

This repository is the main Rosso Labs landing page. The page exists to generate more quote requests for websites and landing pages for companies and solo professionals.

## Priority order

1. conversion
2. clarity of the offer
3. proof and credibility
4. mobile experience
5. technical stability

## Core sections to preserve

- navigation
- hero
- visual showcase
- services
- projects
- commercial banner
- contact
- niche modal

## Copy direction

- direct
- professional
- commercial
- scannable
- benefit-led

## CTA direction

Prefer:

- `Ver modelos e preços`
- `Pedir orçamento`
- `Falar no WhatsApp`
- `Quero meu site agora`

Avoid weak or institutional CTAs.

## Sensitive shared files

- `assets/css/main.css`: shared layout and niche-page responsive behavior
- `assets/js/index.js`: navbar, intro, modal, theme behavior, inline compatibility helpers
- `assets/js/mobile-fix.js`: mobile safety patches
- niche pages under `pages/sitePara*.html`
